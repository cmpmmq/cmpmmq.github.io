markdown & HTML

之前的 blog 是用md写的，md本身也可以另存为 html, docx, pdf 之类的文件。md 本身让我感觉是 HTML 的简化版，不用打 tag，用 # * 之类的符号加粗斜体设置层级，不用渲染格式，和没有 CSS 的 html 文件有类似之处。所以这篇 html 博客也能和谐地渲染成 Keep Theme 主题。
